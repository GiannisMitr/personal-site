Object Detection & Person Segmentation in the Browser

No webcam support found for this device. If you are using an IOS device try using Safari browser.

This module runs inference in your browser using TensorFlow.js library and web camera images as input.
By clicking "Enable Camera", access to your camera will be requested.
Images are NOT sent to a server or stored in any way.
You can review the source code of the page here.

  

Uses a COCO-SSD model ported for TensorFlow.js library. As an encoder it uses a lite version of MobileNetV2 and an SSD network (Single Shot Detector) as the decoder. The model outputs predictions for 90 classes of the COCO dataset and coordinates for the predicted boxes. The predictions are drawn to a canvas element overlayed over the video.
You can view the code here.

Uses a BodyPix model ported for TensorFlow.js library. It uses a MobileNetV1 with an algorithm described here that outputs a segmentation mask for 24 body parts. The mask is drawn using helper functions provided with the model. You can view the code here.
